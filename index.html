Quick Setup

  1. Download ModLoaderH.zip and extract it to a short path (e.g., C:\ModLoaderH\).
  2. Run ModLoaderH.exe as Administrator on first launch.
  3. Open your game, attach process if needed, enable toggles.
  4. Drop new trainers/mods into the /trainers folder.
